#`sync` - v 0.1.0

#Installation

##Using `pip`

`pip install sync`

##Get the latest build from the Source

* Clone the repo `git clone https://github.com/harshasrinivas/sync.git`
* Run `python setup.py install`

##Dependencies

* future `pip install future`

#Usage

* Sync directory `dir1` with `dir2` : `sync dir1 dir2`
